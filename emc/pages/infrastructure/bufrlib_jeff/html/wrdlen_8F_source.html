4 C> THIS SUBROUTINE FIGURES OUT SOME IMPORTANT INFORMATION
5 C> ABOUT THE LOCAL MACHINE ON WHICH THE BUFR ARCHIVE LIBRARY SOFTWARE
6 C> IS BEING RUN AND STORES THIS INTO COMMON BLOCK /HRDWRD/. SUCH
7 C> INFORMATION INCLUDES DETERMINING THE NUMBER OF BITS AND THE NUMBER
8 C> OF BYTES IN A MACHINE WORD AS WELL AS DETERMINING WHETHER THE
9 C> MACHINE USES THE ASCII OR EBCDIC CHARACTER SET.
10 C>
11 C> NOTE: IT IS ONLY NECESSARY FOR THIS SUBROUTINE TO BE CALLED ONCE,
12 C> AND THIS IS NORMALLY DONE DURING THE FIRST CALL TO BUFR ARCHIVE
13 C> LIBRARY SUBROUTINE OPENBF. HOWEVER, THE SUBROUTINE DOES KEEP TRACK
14 C> OF WHETHER IT HAS ALREADY BEEN CALLED; THUS, IF IT IS CALLED AGAIN
15 C> LATER BY A DIFFERENT BUFR ARCHIVE LIBRARY SUBROUTINE, IT WILL JUST
16 C> QUIETLY RETURN WITHOUT (RE)COMPUTING ALL OF THE INFORMATION WITHIN
17 C> COMMON BLOCK /HRDWRD/.
18 C>
