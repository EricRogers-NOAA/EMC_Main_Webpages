GIVEN A TABLE B MNEMONIC, THIS SUBROUTINE RETURNS THE ELEMENT NAME AND UNITS ASSOCIATED WITH THAT MNEMONIC.

THIS SUBROUTINE CAN BE CALLED AT ANY TIME FOLLOWING THE CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE OPENBF FOR THE ASSOCIATED LUNIT.

PROGRAM HISTORY LOG: 2014-10-02 J. ATOR – ORIGINAL VERSION 2014-12-10 J. ATOR – USE MODULES INSTEAD OF COMMON BLOCKS

USAGE: CALL NEMDEFS (LUNIT, NEMO, CELEM, CUNIT, IRET ) INPUT ARGUMENT LIST: LUNIT - INTEGER: FORTRAN LOGICAL UNIT NUMBER FOR BUFR FILE NEMO - CHARACTER*(*): TABLE B MNEMONIC

OUTPUT ARGUMENT LIST: CELEM - CHARACTER*55: ELEMENT NAME ASSOCIATED WITH NEMO CUNIT - CHARACTER*24: UNITS ASSOCIATED WITH NEMO IRET - INTEGER: RETURN CODE 0 = NORMAL RETURN -1 = REQUESTED MNEMONIC COULD NOT BE FOUND, OR SOME OTHER ERROR OCCURRED

REMARKS: THIS ROUTINE CALLS: NEMTAB STATUS THIS ROUTINE IS CALLED BY: None Normally called only by application programs
