4 C> THIS SUBROUTINE PACKS A CHARACTER STRING (CHR) CONTAINING
5 C> NCHR CHARACTERS INTO NCHR BYTES OF AN INTEGER ARRAY (IBAY),
6 C> STARTING WITH BIT (IBIT+1). ON OUTPUT, IBIT IS UPDATED TO POINT TO
7 C> THE LAST BIT THAT WAS PACKED. NOTE THAT THERE IS NO GUARANTEE THAT
8 C> THE NCHR CHARACTERS WILL BE ALIGNED ON BYTE BOUNDARIES WHEN PACKED
9 C> WITHIN IBAY.

89  int = ishft(ival(1),nbitw-nbit)

91  msk = ishft( -1,nbitw-nbit)

94  IF(nbt+nbit.GT.nbitw) THEN

102  msk = ishft( -1,2*nbitw-(nbt+nbit))
103  ibay(nwd+1) = irev(ior(iand(irev(ibay(nwd+1)),not(msk)),int))
104  ENDIF
