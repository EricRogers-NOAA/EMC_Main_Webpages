subroutine strbtm (   N,
  LUN

THIS SUBROUTINE STORES INTERNAL INFORMATION IN MODULE BITMAPS IF THE INPUT ELEMENT IS PART OF A BITMAP.

PROGRAM HISTORY LOG: 2016-05-27 J. ATOR – ORIGINAL AUTHOR 2019-05-22 J. ATOR – ADD CONFIRMATION CHECK

USAGE: CALL STRBTM ( N, LUN ) INPUT ARGUMENT LIST: N - INTEGER: SUBSET ELEMENT LUN - INTEGER: I/O STREAM INDEX INTO INTERNAL MEMORY ARRAYS

REMARKS:

THE FOLLOWING VALUES ARE STORED WITHIN MODULE BITMAPS BY THIS SUBROUTINE AND BY SUBPROGRAMS IGETRFEL, MAKESTAB AND TABSUB:

NBTM = number of stored bitmaps for the current subset (up to a maximum of MXBTM)

NBTMSE(I=1,NBTM) = number of "set" entries (i.e. set to a value of 0) in the bitmap

LINBTM = logical variable set to .TRUE. if a bitmap is currently being read for the current subset

ISTBTM(I=1,NBTM) = ordinal position in subset corresponding to first entry of bitmap

ISZBTM(I=1,NBTM) = size of bitmap (i.e. total number of entries, whether "set" or not)

IBTMSE(I=1,NBTM, J=1,NBTMSE(I)) = ordinal positions in bitmap of bits that were "set"; these ordinal positions can range in value from 1 to ISZBTM(I)

LSTNOD = last jump/link table entry that was processed by function IGETRFEL and whose corresponding value type was either numeric or CCITT IA5

LSTNODCT = current count of consecutive occurrences of LSTNOD

NTAMC = number of Table A mnemonics in jump/link table (up to a maximum of MXTAMC) which contain at least one Table C operator with an X value of 21 or greater in their definition; only Table C operators with an X value of 21 or greater are tracked in this module, since all others (e.g. 2-01, 2-02, 2-07) are automatically processed within subroutines TABSUB and TABENT

INODTAMC(I=1,NTAMC) = location of Table A mnemonic within jump/link table

NTCO(I=1,NTAMC) = number of Table C operators (with an X value of 21 or greater) within the definition of the given Table A mnemonic

CTCO(I=1,NTAMC, J=1,NTCO(I)) = Table C operator

INODTCO(I=1,NTAMC, J=1,NTCO(I)) = location of Table C operator within jump/link table

THIS ROUTINE CALLS: BORT IBFMS LSTJPB THIS ROUTINE IS CALLED BY: RCSTPL RDCMPS Normally not called by any application programs.
