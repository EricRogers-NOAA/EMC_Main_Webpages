INTEGER function icbfms ( character*(*)  STR,
  LSTR 
)

THIS FUNCTION TESTS WHETHER THE INPUT CHARACTER STRING IS "MISSING" BY CHECKING IF ALL OF THE EQUIVALENT BITS ARE SET TO 1.

IT IS SIMILAR TO BUFR ARCHIVE LIBRARY FUNCTION IBFMS, EXCEPT THAT IBFMS TESTS REAL*8 VALUES FOR EQUIVALENCE TO THE PARAMETER BMISS, WHEREAS ICBFMS CHECKS THAT ALL EQUIVALENT BITS ARE SET TO 1 AND IS THEREFORE A MORE PORTABLE AND RELIABLE TEST FOR USE WITH CHARACTER STRINGS.

PROGRAM HISTORY LOG: 2012-06-07 J. ATOR – ORIGINAL AUTHOR 2015-03-10 J. WOOLLEN – IMPROVED LOGIC FOR TESTING LEGACY CASES PRIOR TO BUFRLIB V10.2.0 2016-02-12 J. ATOR – MODIFIED FOR CRAYFTN COMPATIBILITY

USAGE: ICBFMS ( STR, LSTR ) INPUT ARGUMENT LIST: STR - CHARACTER*(*): STRING TO BE TESTED LSTR - INTEGER: NUMBER OF CHARACTERS TO BE TESTED WITHIN STR

OUTPUT ARGUMENT LIST: ICBFMS - INTEGER: RETURN CODE: 0 - STR IS NOT "MISSING" 1 - STR IS "MISSING"

REMARKS: THIS ROUTINE CALLS: iupm() THIS ROUTINE IS CALLED BY: rdcmps() rdtree() ufbdmp() ufdump() Also called by application programs.
