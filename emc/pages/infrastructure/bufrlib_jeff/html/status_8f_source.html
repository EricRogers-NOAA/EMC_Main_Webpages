4 C> THIS SUBROUTINE CHECKS WHETHER LOGICAL UNIT NUMBER LUNIT
5 C> (AND ITS ASSOCIATED BUFR FILE) IS CURRENTLY CONNECTED TO THE
6 C> BUFR ARCHIVE LIBRARY SOFTWARE. IF SO, IT RETURNS THE I/O STREAM
7 C> INDEX (LUN) ASSOCIATED WITH THE LOGICAL UNIT NUMBER, THE LOGICAL
8 C> UNIT STATUS INDICATOR (IL), AND THE BUFR MESSAGE STATUS INDICATOR
9 C> (IM) FOR THAT I/O STREAM INDEX. OTHERWISE, IT CHECKS WHETHER THERE
10 C> IS SPACE FOR A NEW I/O STREAM INDEX AND, IF SO, RETURNS THE NEXT
11 C> AVAILABLE I/O STREAM INDEX IN LUN IN ORDER TO DEFINE LUNIT (IL AND
12 C> IM ARE RETURNED AS ZERO, THEY ARE LATER DEFINED VIA CALLS TO BUFR
13 C> ARCHIVE LIBRARY SUBROUTINE WTSTAT IN THIS CASE). IF THERE IS NO
14 C> SPACE FOR A NEW I/O STREAM INDEX, LUN IS RETURNED AS ZERO (AS WELL
15 C> AS IL AND IM) MEANING LUNIT COULD NOT BE CONNECTED TO THE BUFR
16 C> ARCHIVE LIBRARY SOFTWARE. LUN IS USED TO IDENTIFY UP TO "NFILES"
17 C> UNIQUE BUFR FILES IN THE VARIOUS INTERNAL ARRAYS.
