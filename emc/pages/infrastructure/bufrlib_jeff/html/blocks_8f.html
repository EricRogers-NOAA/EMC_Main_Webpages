subroutine blocks ( integer*4, dimension(mwrd)  MBAY,
  MWRD 
)

BLOCKS WILL ADD IEEE FORTRAN TYPE RECORD CONTROL WORDS TO A PURE BUFR RECORD PASSED FROM MSGWRT, IN PREPARATION FOR OUTPUTING THE RECORD TO DISK.

THE DEFAULT OUTPUT TYPE IS PURE (NO CONTROL WORDS), IN WHICH CASE THIS ROUTINE DOES NOTHING. AN APPLICATION CAN SPECIFY THAT EITHER BIG OR LITTLE ENDIAN RECORD CONTROL WORDS ARE TO BE APPENDED TO PURE BUFR RECORDS VIA A PREVIOUS CALL TO SUBROUTINE SETBLOCK.

THE FOLLOWING DIAGRAM ILLUSTRATES IEEE CONTROL WORDS FOUND IN AN UNFORMATTED FORTRAN RECORD CONRTAINING FOUR 4-BYTE WORDS 

ctw1-wrd1-wrd2-wrd3-wrd4-ctw2
|    |    |    |    |    |
0016-aaaa-bbbb-cccc-dddd-0016

CTW1 AND CTW2 CONTAIN A BYTE COUNT FOR THE DATA RECORD THAT THEY ENCLOSE. THEY CAN BE STORED IN EITHER BIG OR LITTLE ENDIAN BYTE ORDERING (NOTE: CTWS ARE ALWAYS 4-BYTE WORDS)

PROGRAM HISTORY LOG: 2012-09-15 J. WOOLLEN – ORIGINAL AUTHOR
