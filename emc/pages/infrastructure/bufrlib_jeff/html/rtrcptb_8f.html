THIS SUBROUTINE RETURNS THE TANK RECEIPT TIME STORED WITHIN SECTION 1 OF THE BUFR MESSAGE IN ARRAY MBAY.

PROGRAM HISTORY LOG: 2013-10-07 J. ATOR – ADAPTED FROM RTRCPT

USAGE: CALL RTRCPT (MBAY,IYR,IMO,IDY,IHR,IMI,IRET) INPUT ARGUMENT LIST: MBAY - INTEGER: *-WORD PACKED BINARY ARRAY CONTAINING BUFR MESSAGE

OUTPUT ARGUMENT LIST: IYR - INTEGER: TANK RECEIPT YEAR IMO - INTEGER: TANK RECEIPT MONTH IDY - INTEGER: TANK RECEIPT DAY IHR - INTEGER: TANK RECEIPT HOUR IMI - INTEGER: TANK RECEIPT MINUTE IRET - INTEGER: RETURN CODE: 0 = normal return -1 = no tank receipt time was present within MBAY

REMARKS: THIS ROUTINE CALLS: IUPB IUPBS01 THIS ROUTINE IS CALLED BY: RTRCPT Also called by application programs.
