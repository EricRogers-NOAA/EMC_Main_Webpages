4 C> THIS SUBROUTINE DUMPS A DETAILED PRINT LISTING OF THE
5 C> CONTENTS OF THE UNPACKED DATA SUBSET CURRENTLY RESIDING IN THE
6 C> INTERNAL ARRAYS ASSOCIATED WITH THE BUFR FILE IN LOGICAL UNIT LUNIT.
7 C> LUNIT MUST HAVE BEEN OPENED FOR INPUT VIA A PREVIOUS CALL TO BUFR
8 C> ARCHIVE LIBRARY SUBROUTINE OPENBF. THE DATA SUBSET MUST HAVE BEEN
9 C> SUBSEQUENTLY READ INTO THE INTERNAL BUFR ARCHIVE LIBRARY ARRAYS VIA
10 C> A CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE READMG OR READERME,
11 C> FOLLOWED BY A CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE READSB (OR VIA
12 C> A SINGLE CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE READNS!). FOR A
13 C> PARTICULAR SUBSET, THE PRINT LISTING CONTAINS EACH MNEMONIC
14 C> ACCOMPANIED BY ITS CORRESPONDING DATA VALUE (INCLUDING THE ACTUAL
15 C> BITS THAT WERE SET FOR FLAG TABLE VALUES!) AS WELL AS OTHER USEFUL
16 C> IDENTIFICATION INFORMATION. THIS SUBROUTINE IS SIMILAR TO BUFR
17 C> ARCHIVE LIBRARY SUBROUTINE UFBDMP EXCEPT THAT IT DOES NOT PRINT
18 C> POINTERS, COUNTERS AND OTHER MORE ESOTERIC INFORMATION DESCRIBING
19 C> THE INTERNAL SUBSET STRUCTURES. EACH SUBROUTINE, UFBDMP AND UFDUMP,
20 C> IS USEFUL FOR DIFFERENT DIAGNOSTIC PURPOSES, BUT IN GENERAL UFDUMP
21 C> IS MORE USEFUL FOR JUST LOOKING AT THE DATA ELEMENTS.
22 C>
23 C> PROGRAM HISTORY LOG:
24 C> 2002-05-14 J. WOOLLEN -- ORIGINAL AUTHOR
25 C> 2003-11-04 J. WOOLLEN -- MODIFIED TO HANDLE PRINT OF CHARACTER
26 C> VALUES GREATER THAN EIGHT BYTES
27 C> 2003-11-04 S. BENDER -- ADDED REMARKS/BUFRLIB ROUTINE
28 C> INTERDEPENDENCIES
29 C> 2003-11-04 D. KEYSER -- MAXJL (MAXIMUM NUMBER OF JUMP/LINK ENTRIES)
30 C> INCREASED FROM 15000 TO 16000 (WAS IN
31 C> VERIFICATION VERSION); UNIFIED/PORTABLE FOR
32 C> WRF; ADDED DOCUMENTATION (INCLUDING
33 C> HISTORY); OUTPUTS MORE COMPLETE DIAGNOSTIC
34 C> INFO WHEN ROUTINE TERMINATES ABNORMALLY
35 C> 2004-08-18 J. ATOR -- ADDED FUZZINESS TEST AND THRESHOLD FOR
36 C> MISSING VALUE; ADDED INTERACTIVE AND
37 C> SCROLLING CAPABILITY SIMILAR TO UFBDMP
38 C> 2006-04-14 J. ATOR -- ADD CALL TO UPFTBV FOR FLAG TABLES TO GET
39 C> ACTUAL BITS THAT WERE SET TO GENERATE VALUE
40 C> 2007-01-19 J. ATOR -- USE FUNCTION IBFMS
41 C> 2009-03-23 J. ATOR -- ADD LEVEL MARKERS TO OUTPUT FOR SEQUENCES
42 C> WHERE THE REPLICATION COUNT IS > 1; OUTPUT
43 C> ALL OCCURRENCES OF LONG CHARACTER STRINGS
44 C> 2012-02-24 J. ATOR -- FIX MISSING CHECK FOR LONG CHARACTER STRINGS

72 C> ENTERS "q" FOLLOWED BY "<enter>" AFTER THE PROMPT, IN WHICH CASE
