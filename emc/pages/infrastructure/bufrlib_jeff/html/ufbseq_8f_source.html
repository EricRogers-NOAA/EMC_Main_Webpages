4 C> THIS SUBROUTINE WRITES OR READS SPECIFIED VALUES TO OR FROM
5 C> THE CURRENT BUFR DATA SUBSET WITHIN INTERNAL ARRAYS, WITH THE
6 C> DIRECTION OF THE DATA TRANSFER DETERMINED BY THE CONTEXT OF
7 C> ABS(LUNIN) {I.E., IF ABS(LUNIN) POINTS TO A BUFR FILE THAT IS OPEN
8 C> FOR INPUT, THEN DATA VALUES ARE READ FROM THE INTERNAL DATA SUBSET;
9 C> OTHERWISE, DATA VALUES ARE WRITTEN TO THE INTERNAL DATA SUBSET}.
10 C> THE DATA VALUES CORRESPOND TO A SEQUENCE OF TABLE B MNEMONICS WHICH
11 C> ARE REPRESENTED BY A SINGLE TABLE A OR TABLE D SEQUENCE MNEMONIC.
12 C> THIS SEQUENCE MNEMONIC MAY ITSELF CONTAIN ONE OR MORE TABLE D
13 C> SEQUENCE MNEMONICS ALONG WITH TABLE B MNEMONICS, THE SEQUENCE
14 C> MNEMONICS HERE CAN USE EITHER DELAYED REPLICATION, REGULAR (I.E.,
15 C> NON-DELAYED) REPLICATION OR THEY CAN HAVE NO REPLICATION AT ALL.
16 C> HOWEVER, IN CASES WHERE THIS SUBROUTINE IS WRITING DATA VALUES TO
17 C> SEQUENCES USING DELAYED-REPLICATION, THE APPLICATION PROGRAM MUST
18 C> FIRST CALL BUFR ARCHIVE LIBRARY ROUTINE DRFINI TO PRE-ALLOCATE THE
19 C> SPACE NEEDED TO EXPAND THE DELAYED-REPLICATION SEQUENCE (THE NUMBER
20 C> OF REPLICATIONS IN DELAYED-REPLICATION IS SET TO ZERO BY DEFAULT).
21 C> (SEE BUFR ARCHIVE LIBRARY DRFINI DOCBLOCK REMARKS FOR MORE
22 C> INFORMATION.) IF UFBSEQ IS READING VALUES, THEN EITHER BUFR ARCHIVE
23 C> LIBRARY SUBROUTINE READSB OR READNS MUST HAVE BEEN PREVIOUSLY
24 C> CALLED TO READ THE SUBSET FROM UNIT ABS(LUNIN) INTO INTERNAL
25 C> MEMORY. IF IT IS WRITING VALUES, THEN EITHER BUFR ARCHIVE LIBRARY
26 C> SUBROUTINE OPENMG OR OPENMB MUST HAVE BEEN PREVIOUSLY CALLED TO
27 C> OPEN AND INITIALIZE A BUFR MESSAGE WITHIN MEMORY FOR THIS
28 C> ABS(LUNIN).
29 C>
