4 C> IF DYNAMIC MEMORY ALLOCATION IS BEING USED, THIS ROUTINE
5 C> IS CALLED DURING THE FIRST CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE
6 C> OPENBF TO DYNAMICALLY ALLOCATE MEMORY FOR ALL REQUIRED FORTRAN
7 C> LANGUAGE ARRAYS. THESE ARRAYS ARE SIZED USING VALUES INPUT DURING
8 C> ONE OR MORE PREVIOUS CALLS TO BUFR ARCHIVE LIBRARY SUBROUTINE
9 C> ISETPRM, OR ELSE USING ONE OR MORE DEFAULT VALUES SPECIFIED IN
10 C> MODULE FILES IF ISETPRM IS NEVER CALLED FOR THOSE PARTICULAR SIZE
11 C> VALUES.
12 C>
13 C> MEMORY ALLOCATED WITHIN THIS ROUTINE CAN BE FREED VIA A USER CALL
14 C> TO BUFR ARCHIVE LIBRARY ROUTINE ARDLLOCF (IN CASES WHERE THE
15 C> APPLICATION PROGRAM MAY WISH TO MOVE ON TO OTHER TASKS NOT
16 C> REQUIRING ANY FURTHER CALLS TO BUFR ARCHIVE LIBRARY ROUTINES), OR
17 C> ELSE IT WILL BE FREED AUTOMATICALLY WHEN THE APPLICATION PROGRAM
18 C> TERMINATES.
19 C>
20 C> PROGRAM HISTORY LOG:
21 C> 2014-12-04 J. ATOR -- ORIGINAL AUTHOR

23 C> AND MODA_NRV203
