4 C> THIS SUBROUTINE SHOULD ONLY BE CALLED WHEN LOGICAL UNIT
5 C> LUNIT HAS BEEN OPENED FOR OUTPUT OPERATIONS. IT PACKS UP THE
6 C> CURRENT SUBSET WITHIN MEMORY AND THEN TRIES TO ADD IT TO THE
7 C> BUFR MESSAGE THAT IS CURRENTLY OPEN WITHIN MEMORY FOR THIS LUNIT.
8 C> THE DETERMINATION AS TO WHETHER OR NOT THE SUBSET CAN BE ADDED TO
9 C> THE MESSAGE IS MADE VIA AN INTERNAL CALL TO ONE OF THE BUFR ARCHIVE
10 C> LIBRARY SUBROUTINES WRCMPS OR MSGUPD, DEPENDING UPON WHETHER OR NOT
11 C> THE MESSAGE IS COMPRESSED. IF IT TURNS OUT THAT THE SUBSET CANNOT
12 C> BE ADDED TO THE CURRENTLY OPEN MESSAGE, THEN THAT MESSAGE IS
13 C> FLUSHED TO LUNIT AND A NEW ONE IS CREATED IN ORDER TO HOLD THE
14 C> SUBSET.
15 C>
