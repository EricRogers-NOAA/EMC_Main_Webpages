4 C> GIVEN A MNEMONIC WITHIN A SUBSET CURRENTLY OPEN FOR
5 C> READING, THIS SUBROUTINE CHECKS WHETHER THE MNEMONIC REFERENCES
6 C> ANOTHER ELEMENT WITHIN THE SAME SUBSET VIA AN INTERNAL BITMAP, AND
7 C> IF SO RETURNS THE REFERENCED MNEMONIC AND ITS LOCATION. IF THERE
8 C> IS MORE THAN ONE OCCURRENCE OF THE INPUT MNEMONIC WITHIN THE OVERALL
9 C> SUBSET DEFINITION, THE SUBROUTINE WILL RETURN THE MNEMONIC REFERRED
10 C> TO BY THE (NTAGI)th OCCURRENCE OF THE INPUT MNEMONIC, COUNTING FROM
11 C> THE BEGINNING OF THE OVERALL SUBSET DEFINITION. A SUBSET DEFINITION
12 C> MUST ALREADY BE IN SCOPE, EITHER VIA A PREVIOUS CALL TO BUFR ARCHIVE
13 C> LIBRARY SUBROUTINE READSB OR EQUIVALENT.
14 C>
15 C> PROGRAM HISTORY LOG:
16 C> 2016-06-07 J. ATOR -- ORIGINAL AUTHOR
17 C>

19 C> INPUT ARGUMENT LIST:
20 C> LUNIT - INTEGER: FORTRAN LOGICAL UNIT NUMBER FOR BUFR FILE
21 C> TAGI - CHARACTER*(*): MNEMONIC
22 C> NTAGI - INTEGER: ORDINAL OCCURRENCE OF TAGI FOR WHICH TAGRE
23 C> IS TO BE RETURNED, COUNTING FROM THE BEGINNING OF
24 C> THE OVERALL SUBSET DEFINITION
25 C>
26 C> OUTPUT ARGUMENT LIST:
