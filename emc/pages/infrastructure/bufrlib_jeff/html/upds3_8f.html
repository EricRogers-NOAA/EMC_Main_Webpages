THIS SUBROUTINE UNPACKS AND RETURNS THE DESCRIPTORS CONTAINED WITHIN SECTION 3 OF A BUFR MESSAGE STORED IN ARRAY MBAY.

THE START OF THE BUFR MESSAGE (I.E. THE STRING "BUFR") MUST BE ALIGNED ON THE FIRST FOUR BYTES OF MBAY. NOTE ALSO THAT THIS SUBROUTINE DOES NOT RECURSIVELY RESOLVE SEQUENCE DESCRIPTORS THAT APPEAR WITHIN SECTION 3; RATHER, WHAT IS RETURNED IS THE EXACT LIST OF DESCRIPTORS AS IT APPEARS WITHIN SECTION 3.

USAGE: CALL UPDS3 (MBAY, LCDS3, CDS3, NDS3) INPUT ARGUMENT LIST: MBAY - INTEGER: *-WORD PACKED BINARY ARRAY CONTAINING BUFR MESSAGE LCDS3 - INTEGER: DIMENSIONED SIZE (IN INTEGER WORDS) OF CDS3; USED BY THE SUBROUTINE TO ENSURE THAT IT DOES NOT OVERFLOW THE CDS3 ARRAY

OUTPUT ARGUMENT LIST: CDS3 - CHARACTER*6: *-WORD ARRAY CONTAINING UNPACKED LIST OF DESCRIPTORS (FIRST NDS3 WORDS FILLED) NDS3 - INTEGER: NUMBER OF DESCRIPTORS RETURNED
